# Guide to running notebook

1. Clone/Download the repo 
2. ```cd``` into the folder
3. Make the python virtual enviroment
   ```
   python -m venv venv 

   source venv/bin/activate 

   pip install -r requirements.txt
   ```
1. Run the notebook
   ``` 
   jupyter notebook main.ipynb 
   ```
